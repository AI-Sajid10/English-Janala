# 📚 English Janala

A modern and responsive English learning platform built using HTML, CSS, and JavaScript.  
This project helps users learn English vocabulary interactively with categorized lessons, pronunciation support, responsive UI, and dynamic API integration.

🌐 Live Site: https://english-janala-by-sajid.netlify.app/
📂 GitHub Repository: https://github.com/AI-Sajid10/English-Janala

---

## ✨ Features

### 🔐 Authentication UI
- Login interface with smooth user experience
- Responsive navbar and navigation system

### 📖 Vocabulary Learning System
- Dynamic vocabulary cards
- Word meaning and pronunciation support
- Lesson-wise vocabulary loading
- Interactive learning experience

### 🔊 Pronunciation Support
- Text-to-Speech API integration
- Users can hear correct pronunciation instantly

### 📱 Fully Responsive Design
- Mobile
- Tablet
- Desktop friendly layout

### 🎨 Modern UI/UX
- Clean and minimal design
- Smooth hover effects and transitions
- Beautiful typography and spacing

### ⚡ API Integration
- Dynamic data fetching using JavaScript Fetch API
- Real-time vocabulary rendering

### ❓ FAQ Section
- Frequently asked questions section
- Accordion functionality

### ⏳ Loading Spinner
- Loading animation while fetching data

### 🚫 Error Handling
- Handles unavailable or missing data properly
